INICIO HTML


Oque o HTML 5 ? O html e a quinta e ultima versão do HTML, sigla de Hypertext Markup Language, uma das primeiras e principais linguagens de marcação da web. O HTML serve, basicamente, para o desenvolvimento de paginas com conteúdos simples como textos,videos,imagens,audio etc.

TAGS HTML 5

As tags HTML são elementos usados para marcar e formatar conteudos, basicamente sem elas nao teria como criar um site.Cada tag HTML tem um nome e uma sintaxe que determina como ela deve ser aberta ou fechada.
AQUI ESTA ALGUMAS TAGS MAIS USADAS:
< p > < / p > usadas para criar para criar parágrafos
< head >< / head > para informação sobre documentos
< body >< / body > o conteudo do seu site deve ficar dentro dessa tag.
< br > usada para criar quebras de linhas
< img > para adicionar imagens no site < h1 > ao < h6 > para criar titulos ou subtitulos dentro do conteudo

IMAGENS NO HTML

As imagens no seu site são muito importante para manter o cliente navegando nele e tambem deixar o site bem mais profissional e bonito também então dependo de que se trate o assunto do seu site e muito importante o uso de imagens.
