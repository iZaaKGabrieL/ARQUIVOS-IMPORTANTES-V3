JAVA E JAVASCRIPT



Muitas pessoas acham que as linguagens java e javascript são iguais ou que se tratam da mesma linguagem apenas pelo nome mais a verdade e que, com o sucesso inicial do JavaScript, a mudança do nome de LiveScript para JavaScript foi inevitável, e, com certeza, veio por influencia da própria SUN, que mantém uma linguagem de programação chamada JAVA. E claro que as linguagens de programação JAVA e JAVASCRIPT são parecidas somente no nome, já que se diferem no conceito e no uso.

CRIAÇÃODO JSCRIPT



A Microsoft, vendo tudo isso que estava acontecendo com o javaScript, percebeu que era uma boa ideia e, em vez de ajudar no desemvolvimento do JavaScript e se tornar parceira desta linguagem, resolveu seguir um caminho diferenteao JavaScript e acabou criando o JScript para rodar somente no internet Explorer, o que causou uma grande dor de cabeça aos desenvolvedores WEB por volta de 1997.
Ja uma outra empresa chamada de ECMA , vendo que o JScript so rofava no internet Explorer, acabou criando o ECMAscript, eles resoulveram padronizar a linguagem de forma que ela pudesse funcionar de maneira geral em todos os computadores entao foi assim que nasceu uma linguagem de programação chamada ECMAScript.

ECMSCRIPT VIROU JAVASCRIPTT


Sendo realista todos os progamas atualmente desenvolvido usando JavaScript sao na verdade progamas em ECMAScript. No entanto, por motivos de marketing e pelo motivo do nome "JAVASCRIPT", ja esta muito consolidado, o nome permaneceu JavaScript, em todos os lugares, como livros, revistas, sites e outros.
